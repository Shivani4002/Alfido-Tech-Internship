# 🚀 Projects Overview 

---

## 📌 1. Customer Behavior Analysis

📊 Dataset:  
https://www.kaggle.com/datasets/bhanupratapbiswas/customer-behavior-analysis

### 🎯 Objective
Analyze customer transactions to identify:
- Customer segments  
- Purchase patterns  
- Churn risks  

### 🔧 Work Done
- Data cleaning and preprocessing  
- Feature engineering (RFM Analysis)  
- Customer segmentation using clustering techniques  
- Analysis of customer behavior  
- Business insights and recommendations  

### 📌 Outcome
Identified high-value customer segments and improved understanding of customer retention patterns.

---

## 📌 2. Sales Performance Analysis

📊 Dataset:  
https://www.kaggle.com/datasets/bhanupratapbiswas/superstore-sales

### 🎯 Objective
Analyze sales performance across:
- Regions  
- Categories  
- Time periods  

### 🔧 Work Done
- Data cleaning and preparation  
- KPI creation (Revenue, Profit, Profit Margin, AOV)  
- Regional and category analysis  
- Trend and seasonality analysis  
- Power BI dashboard development  

### 📌 Outcome
Delivered insights on sales performance and profitability with interactive dashboards.

---

## 📌 3. Website Traffic Analysis

📊 Dataset:  
https://www.kaggle.com/datasets/bhanupratapbiswas/website-traffic-analysis

### 🎯 Objective
Analyze website traffic logs to understand:
- User behavior  
- Bounce rates  
- Traffic sources  
- Content performance  

### 🔧 Work Done
- Data cleaning and preprocessing  
- Session and user behavior analysis  
- Bounce rate calculation  
- Traffic trend visualization  
- Engagement pattern analysis  

### 📌 Outcome
Generated insights to improve engagement and conversion rates.
